# 🛒 Ghosia Mini Market - Frontend

React + Vite frontend for Ghosia Mini Market Bristol.

## Branches
| Branch | Purpose |
|---|---|
| `main` | Stable production code |
| `feature/ui-improvements` | UI and design changes |
| `feature/cart` | Cart and checkout features |
| `feature/search-filter` | Search and category filter |
| `feature/animations` | Transitions and animations |

## How to Run
```bash
npm install
npm run dev
```
Runs on http://localhost:5173

## Tech Stack
- React 18
- Vite 5
- Pure CSS (no frameworks)
